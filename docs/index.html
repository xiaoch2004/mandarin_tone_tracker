Madanrin Tone Tracker

Plot tone contour for mandarin learners/普通话声调轮廓绘制

Introduction

The mandarin_tone_tracker plots mandarin tone contour for mandarin learners. The code use crepe to extract pitch. It also adds these post-processing methods to adapt mandarin frequency characteristics:

  • Eliminate frames with too-low frequencies (default threshold 100.0Hz)
  • Eliminate frames with too-low confidence (default threshold 0.40)
  • Eliminate frames with too-low energy (default threshold 5e-4)
